Software Preferences

This is a list of all the software I currently use together with any relevant customization. I wrote this just in case my computers spontaneously combust and I need to onboard new devices, and for anyone interested in my software. The order given is purposeful, and later sections build on the earlier ones. I’ll continuously update this document as my software preferences grow and evolve.

Note: I found a good amount of this software through an MIT course called The Missing Semester of Your CS Education, which I highly recommend checking out.


OS

I use macOS Big Sur and run Parallels as a VM for Windows 10 and Ubuntu 20.04. I’ve only been using Linux for 1.5 years so I haven’t explored other distros in depth (although I’ve played around with CentOS and Kali a bit).

Terminal

I use Alacritty because it's GPU accelerated, has great integration with tmux and Vim, and is visually appealing. tmux is a must have for terminal productivity. I also like to use shellCheck to lint shell scripts.

zsh

My main shell is zsh. I use Oh My Zsh because its fucking awesome, has great themes (I use fino-time because its prompt optimizes for the amount of information I can see while maintaining simplicity, and it takes up a whole line which makes it easy to scroll through shell history and find my cursor), and the plugins of course. zsh-syntax-highlighting, zsh-history-substring-search, zsh-autosuggestions and fasd are must haves for optimizing command line efficiency. tldr-pages are useful for learning how to use commands when the man pages get annoying.

Styling

Black is a necessity for formatting Python. It is a great tool precisely because it is mostly unconfigurable. I use Prettier to format my web dev files (HTML, CSS, JS).

Vim

Vim is my main text editor and an essential part of my workflow. I recommend learning to use it thoroughly. You can start by completing the built-in vimtutor by running vimtutor from the command line. Also, if you run macOS, a version of Vim comes preinstalled, but you should use the homebrew version instead because the preinstalled Vim doesn’t come with some dependencies (namely python3) needed to support important plugins.

For plugins, I use both vim-plug and Vundle, since some plugins are only compatible with one installer. I write all of my LaTeX in Vim using VimTeX, and my default pdf previewer is zathura. To write LaTeX painlessly and quickly, I use UltiSnips. Read this article to learn how to optimize the aforementioned setup. To navigate my filesystem in Vim, I use nerdtree. I also have trouble with the default undo function in Vim, so I use undotree. Checkout VimAwesome to find cool plugins.

Filesystem

I use FileVault to encrypt the startup disks on my Macs. I primarily use locate with updatedb as my file finder, though fd is a good alternative if your computer runs slow using updatedb (though I find the speed of locate worth the constant updating of updatedb). less is more for opening large files. If I was less familiar with my OSs (maybe if I were working on a remote), then I might find broot and fzf more useful.

Workflow

Creating good keybinds is essential for optimizing productivity. I use Karabiner Elements, Hammerspoon, Vimium, and configure macOS settings for my keybinds (note that I try to keep these as consistent as possible across my OSs).For macOS, I flip the esc and caps lock, flip ctrl and command, use Ctrl-leftarrow/rightarrow to switch between desktops, close applications with CMD+Q, and sleep the display by moving the mouse to the top corner. I use standard Vimium keybinds (but remember that CMD+D bookmarks a page in Chrome!). I only use Karabiner for the F keys. You can see my Hammerspoon config in my dotfiles, though some notable keybinds are CMD+Return to open Alacritty, CMD+Alt+S to open Spotify, and CMD+Alt+Ctrl+leftarrow/rightarrow to tile my screen.

dotfiles

I use Dotbot and symlinks to manage my dotfiles. This is a must have for maintaining consistency and being productive across multiple devices (and OSs). Some great reading about dotfiles can be found here.

Browser

I use Chrome because its fast and heavily supported. I've already mentioned that I use Vimium; I started using it primarily to get better at Vim, but I stuck with it because it feels natural now (and mice suck). GoFullPage allows me to screencapture entire websites. If yuo're not using ADBlock Plus, do you even live in the 21st century? LastPass and computer generated passwords are ESSENTIAL for security, and HTTPS Everywhere helps me browse seafer. I use Boomerang to schedule emails. Grammarly ensirs i can grammer, and Power Thesaurus enables me to dazzle you with piquant sentences. Honey is a Krona knockoff for the web, and Lustre helps you find the best of everything. EndNote makes it much easier to find and read academic PDFs by linking with your .edu email. Lastly, JSONView and React Developer Tools help me develop on the web.
